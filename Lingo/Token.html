    class Lingo::Token

    Die Klasse Token, abgeleitet von der Klasse StringA, stellt den Container für ein einzelnes Wort eines Textes dar. Das Wort wird mit einem Attribut versehen, welches der Regel entspricht, die dieses Wort identifiziert hat.

    Steht z.B. in ruby.cfg eine Regel zur Erkennung einer Zahl, die mit NUM bezeichnet wird, so wird dies dem Token angeheftet, z.B. Lingo::StringA.new(‘100’, ‘NUM’) -> #100/NUM#
