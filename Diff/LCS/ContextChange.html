    class Diff::LCS::ContextChange

    Represents a contextual change. Contains the position and values of the elements in the old and the new sequenced enumerables as well as the action taken.

    Attributes

    action[R]

    Returns the action this Change represents. Can be ‘+’ (Diff::LCS::ChangeTypeTests#adding?), ‘-’ (Diff::LCS::ChangeTypeTests#deleting?), ‘=’ (Diff::LCS::ChangeTypeTests#unchanged?), # or ‘!’ (Diff::LCS::ChangeTypeTests#changed?). When created by Diff::LCS#diff or Diff::LCS#sdiff, it may also be ‘>’ (Diff::LCS::ChangeTypeTests#finished_a?) or ‘<’ (Diff::LCS::ChangeTypeTests#finished_b?).

    Simplifies a context change for use in some diff callbacks. ‘<’ actions are converted to ‘-’ and ‘>’ actions are converted to ‘+’.

    # File lib/diff/lcs/change.rb, line 149
def self.simplify(event)
  ea = event.to_a

  case ea[0]
  when '-'
    ea[2][1] = nil
  when '<'
    ea[0] = '-'
    ea[2][1] = nil
  when '+'
    ea[1][1] = nil
  when '>'
    ea[0] = '+'
    ea[1][1] = nil
  end

  Diff::LCS::ContextChange.from_a(ea)
end

def <=>(other)
  r = @action <=> other.action
  r = @old_position <=> other.old_position if r.zero?
  r = @new_position <=> other.new_position if r.zero?
  r = @old_element <=> other.old_element if r.zero?
  r = @new_element <=> other.new_element if r.zero?
  r
end
